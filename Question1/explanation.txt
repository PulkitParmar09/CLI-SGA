1.User Identity Verification

 whoami = used this command to display the currently logged-in user.
Then,used groups command to see all the groups that my user account belongs to.

2. Workspace Validation

pwd = used this command to display the current working directory.
Then I used ls -l to list all files and directories in long format, showing permissions, ownership, and file size.

3. Environment Confirmation File

I created a file named user_info.txt using the echo command and wrote the text “Linux user environment verified” inside it.

4. File Integrity Check

wc -m  used this command to count the total number of characters present in the user_info.txt file.

5. Learning the Tools

man mkdir = used this command to access the manual page of mkdir command.
The -p option allows creating parent directories if they do not already exist.

6. Home Directory Inspection

ls ~  = used this command to list all files and directories in my home directory in alphabetical order.

7. Log Investigation

 grep "admin" log.txt = used this  command to search for the word “admin” inside the log.txt file and display only the matching lines.

8. System Information Check

 uname -r = used this command to display the Linux kernel version currently running on the system.

9. Network Connectivity Test

 ping -c 4 www.google.com = used this command to send ICMP packets to Google and verify that the network connection is working.

10. System Health Awareness

 uptime = used this command to check how long the system has been running, the number of logged-in users, and the system load average.
